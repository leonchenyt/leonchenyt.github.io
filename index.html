Mechaton

January 2019 - April 2019

Mechaton is a Co-op tactical twin stick shooter with randomly generated levels and loot based progression system.

I focus on the split viewport system for multiple players and the particle system. The viewports keep track of the distance between four players and decide whether the viewports should be merged or split.

Pathfinding

October 2020
Pathfinding

Implemented Astar pathfinding algorithm with generic heap as open list container on a grid terrain. Generic heap uses unordered_map as internal lookup table for the element's index of the heap.

Rubberbanding and smoothing are applied to the final path result. I also implemented Floyd-Warshall to preprocess the grid as an extra practice.
